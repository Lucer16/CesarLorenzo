# 📊 Landing Page - Servicios Contables Cesar Lorenzo (Panamá)

Una solución web de una sola página (**One Page**) diseñada para contadores y consultores financieros que buscan una presencia digital moderna, profesional y de alta conversión.

## ✨ Características Principales
- **Diseño Premium:** Estética *Dark Mode* con acentos en azul eléctrico y efectos de *Glassmorphism*.
- **Optimizado para Panamá:** Secciones específicas para DGI (Renta), CSS (SIPE), ITBMS y Avisos de Operación.
- **Enfoque en Conversión:** Botones estratégicos para contacto directo por **WhatsApp** y llamadas telefónicas.
- **Tecnología Moderna:** Construido 100% con **Tailwind CSS** y micro-animaciones (Animate.css).
- **Responsive Design:** Totalmente adaptable a dispositivos móviles y tablets.

## 🛠️ Tecnologías Utilizadas
* **HTML5** - Estructura semántica.
* **Tailwind CSS** - Framework de diseño utilitario para estilos personalizados.
* **FontAwesome** - Iconografía profesional.
* **Animate.css** - Micro-interacciones y animaciones de entrada.
